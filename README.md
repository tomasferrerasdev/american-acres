## Tech Stack

**Client:** React, SASS, Framer Motion, React reveal


## Run Locally

Clone the project

```bash
  git clone https://github.com/tomasferrerasdev/american-acres.git
```

Go to the project directory ( remember to delete node_modules )

```bash
cd my-project
```

Install dependencies

```bash
yarn install
```
or simply
```bash
yarn
```

Start the server

```bash
yarn start
```
